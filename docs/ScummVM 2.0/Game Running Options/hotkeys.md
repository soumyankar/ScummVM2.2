# Hotkeys

ScummVM supports various in-game hotkeys.

- Common Hotkeys:
```
Hotkeys!
```
- ScummVM Hotkeys:
```
Hotkeys!
```
- Game Specific Hotkeys:
	- LucasArts Games:
	```
	Hotkeys!	
	```
	- Revolution Software Games:	
	```
	Hotkeys!	
	```
	- ..and others.
